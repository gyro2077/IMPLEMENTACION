# ADR-002: Por qué Docker Compose primero

## Estado
Aceptado

## Contexto
El MVP debe ser demostrable rápido y con baja complejidad operacional.

## Decisión
Usar Docker Compose en Fase 1 para levantar todos los servicios locales.

## Consecuencias

- Menos carga cognitiva que iniciar con Kubernetes.
- Ciclo de prueba rápido para iterar.
- Fácil de enseñar en una demo técnica.
